# GIT_INFO
Показать методы списков из уроков

Параметр E e означает, что метод принимает одно значение любого из типов.

add(E e) — добавить значения в список. В отличие от массивов, индекс элемента можно не указывать — в этом случае новый элемент добавляется в конец списка.

add(int index, E e) — добавить значение на конкретную позицию. Как и в массивах, индексация в списках начинается с нуля. Если по этому индексу уже хранится какой-то элемент, он и все остальные подвинутся, а новый займёт его позицию.

get(int index) — получить элемент из списка.

size() — узнать количество элементов списка. Размер списка равен количеству сохранённых в него значений.

remove(int index) — удалить элемент из списка по индексу.

remove(Object o) — удалить элемент из списка по значению.

clear() — удалить все элементы списка.

isEmpty() — проверить, есть ли в списке элементы. Возвращает значение булева типа: true — если список пустой, false — если в нём содержится хотя бы один элемент.

contains(E e) — проверить, добавлен ли элемент в список. Результат вызова можно сохранить в булеву переменную или сразу подставить в условие ветвления.

Показать методы хеш-таблиц из уроков

put(K key, V value) — добавить элементы. Принимает любые типы. В качестве первого аргумента передаётся ключ, а в качестве второго — значение.

get(Object key) — получить элементы из хеш-таблицы.

remove(Object key) — удалить элемент по ключу.

clear() — очистить хеш-таблицу полностью.

containsKey(Object key) — проверить наличие ключа.

containsValue(Object value) — проверить наличие значения.
